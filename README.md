# Isaac Garee Academic Pages Starter

This is a compact starter version of an Academic Pages-style GitHub Pages site for `isgaree.github.io`.

## Upload steps

1. Delete everything currently inside your `isgaree.github.io` repository folder, especially `.nojekyll` if it exists.
2. Unzip this folder.
3. Copy the contents of this folder into the root of your local `isgaree.github.io` repo.
4. Commit and push with GitHub Desktop.
5. On GitHub, use **Settings → Pages → Deploy from a branch → main / root**.

## Important files to edit

- `_config.yml` controls your name, URL, sidebar photo, email, GitHub, LinkedIn, and site settings.
- `_pages/about.md` controls the homepage.
- `_pages/research.md` controls the research page.
- `_pages/cv.md` controls the CV page.
- `_data/navigation.yml` controls the top menu.
- `images/profile.jpg` is your sidebar photo.
- `files/Isaac_Garee_Resume_2026.pdf` is the résumé PDF.

## Notes

Do not add a `.nojekyll` file. This site uses Jekyll and the Academic Pages remote theme.
