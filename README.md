# isgaree.github.io

A lightweight static personal academic website for Isaac Garee.

## Files

- `index.html` — homepage
- `research.html` — research projects
- `bio.html` — education, interests, and contact information
- `assets/site.css` — site styling
- `assets/profile.jpg` — profile photo
- `assets/Isaac_Garee_Resume_2026.pdf` — linked resume PDF
- `.nojekyll` — tells GitHub Pages to serve the site as plain static files

## How to publish

1. Create or open the GitHub repository named `isgaree.github.io`.
2. Upload the contents of this folder to the root of that repository. Do not upload the folder as a nested subfolder.
3. Commit the files to the `main` branch.
4. In GitHub, go to **Settings → Pages** and choose **Deploy from a branch**, then select `main` and `/root`.
5. After GitHub finishes publishing, the site should appear at `https://isgaree.github.io`.

## Editing notes

- Update links in the sidebar inside each HTML file.
- Replace `assets/profile.jpg` to change the profile photo.
- Remove the resume PDF link if you do not want the resume publicly downloadable.
